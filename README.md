# GoCart Web

## Included
- register, login, me, logout, google login service hook
- products, categories, product detail, reviews
- cart item add/update/delete services
- wishlist add/delete services
- orders list/detail/create services
- coupons validate/apply services
- notifications list/mark read/mark all
- inventory admin services
- shipping, payments, addresses generic service layer

## Run
```bash
cp .env.example .env.local
npm install
npm run dev
npm run build
```
